スウィングトレード（1週間〜1ヶ月）向け・自動分析システム

🗺️ システム全体像

毎朝7時、AIが以下を自動で実行してメールで届けます。

🌐 データ取得
マクロ指標 + 全銘柄の株価
VIX・ドル円・NYダウ・S&P500・日経平均・SOX指数を取得。ウォッチリスト25銘柄のRSI・移動平均・出来高を計算。
📚 履歴参照
過去7日間のアドバイスを確認
先週どの銘柄を推奨したか・その後の値動きを追跡。一貫性のある判断のために使用。
🤖 AI分析
Claudeが売買判断を作成
マクロ環境・テクニカル・過去履歴を総合して「買い/継続/売り」を判断。エントリー価格・目標値・損切り価格まで明示。
📧 メール
分析結果をメールで送信
損切りアラートがある場合は冒頭に緊急表示。

📁 ファイル構成
  • data/portfolio.json 現金・保有銘柄を管理。Webページから編集可能。手動
  • data/watchlist.json 毎日分析する25銘柄リスト。VS Codeで編集。手動
  • data/universe.json 月次入れ替えの候補50銘柄。通常は触らない。手動
  • data/history.json 過去7日分の分析履歴。ボットが自動更新。自動
  • main.py ボットのメインプログラム。触らない。
  • .github/workflows/main.yml 毎朝7時に自動実行するスケジュール設定。
✏️ よくある操作

株を買ったとき

1
Webページを開く
ブックマークから「ポートフォリオ管理」を開く
2
「＋ 銘柄を追加」をタップ
銘柄コード・銘柄名・株数・購入単価を入力
3
現金残高を更新して「保存する」
購入金額分を現金から引いた額に変更

株を売ったとき

1
Webページで該当銘柄を「削除」
保有銘柄一覧から削除ボタンをタップ
2
現金残高に売却額を加算して「保存する」
売却金額を現金に足した額に更新

ウォッチリストを変えたいとき

1
VS Codeで data/watchlist.json を開く
左のファイル一覧から選択
2
銘柄を追加・削除して保存（Ctrl+S）
形式: {"code": "7203", "name": "トヨタ自動車", "sector": "自動車"}
3
VS Codeの枝マーク → コミット → 同期
GitHubに反映されて翌朝から新リストで分析
🔄 月次ウォッチリスト入れ替え

毎月の第1営業日（月初め最初の平日）に、通常メールに加えて「月次ローテーション提案」メールが届きます。

AIが50銘柄のユニバースから「外すべき銘柄」「追加すべき銘柄」を提案します。最終判断はあなたが行い、気に入った提案だけをwatchlist.jsonに反映してください。
📈 テクニカル指標の見方
  • RSI 30以下=売られすぎ（買い候補）、40〜65=スウィング買い適正、70以上=過熱（要注意）
  • MA25/MA75 MA25＞MA75なら中期上昇トレンド。MA25がMA75を下回ったら要注意（デッドクロス）
  • VIX 20以下=平常、20〜25=警戒、25超=新規買い禁止。「恐怖指数」とも呼ばれる
  • 出来高比率 1.5以上=方向感のある動き。0.5以下=様子見相場。1.0が通常
🛡️ 自動適用されているリスク管理ルール
  • 損切りライン 買値から-8%でアラート。メール冒頭に「緊急」表示
  • 1銘柄上限 総資産の15%まで。超える場合は1株単位（ミニ株）を推奨
  • VIX連動 VIX25超は新規買い推奨なし。相場全体が不安定な時は現金維持
  • 最低保有期間 買ってから5営業日以内の売りは原則禁止（明確な損切りを除く）
❓ よくある質問
メールが届かない日がある
GitHubの Actions タブ でエラーが出ていないか確認してください。赤い×が付いていればエラーです。
ウォッチリストにない銘柄が分析される
保有銘柄がウォッチリストに含まれていない場合、自動的に追加取得します。watchlist.jsonに追加しておくと管理しやすいです。
スマホのWebページが開けなくなった
GitHub Personal Access Token の有効期限が切れている可能性があります。こちらで再発行して「別のトークンに切り替える」から入力しなおしてください。
「常時」とボタンを押したが大丈夫か
問題ありません。VS Codeがステージング操作を自動化するだけで、GitHubへの送信（同期）は引き続き手動で行います。
銘柄コードの調べ方は？
Yahoo!ファイナンス（finance.yahoo.co.jp）で銘柄名を検索すると4桁のコードが確認できます。
💰 API使用状況ダッシュボード

Claude APIのトークン使用量とコストを確認できます。ポートフォリオページでトークンを設定済みの場合は自動読み込みします。
